# woman
man-pages like cli utility for based.cooking

# Example Usage
To view a recipe:
```
$ woman Arroz Chaufa
```

To list the names of all available recipes:
```
$ woman --list
```

# Tab Completion
Tab completion is available (currently only in bash)

# Dependencies
Install `pandoc` before using.

# Install
```
$ sudo ./install
```

# Uninstall
```
$ sudo ./uninstall
```
